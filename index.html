What is DevOps?

A software development method that stresses communication, collaboration and integration between software developers and information technology (IT) professionals.
Simply put, it's a way to work together and automate repeatable tasks

How does it fit into full-stack development

Continuous Integration

The practice of integrating multiple branches and features of code into a single golden pre-production repository branch that I like to call the “develop” branch

Continuous Development

Working in an environment that allows you take on tasks, well, continuously without being blocked by other developers, technical roadblocks, etc.

Continuous Deployment (or Delivery)

The developer and their pairs or team leads have done the due diligence to deploy code not just to the develop or QA environment... but all the way to production

Agile + DevOps

The idea of DevOps came from the need to keep up with increased software velocity achieved through agile methods

Agile methodologies involve ongoing planning, continuous testing, continuous integration, and other endless forms for both the project and software
